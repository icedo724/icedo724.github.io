# 안녕하세요, 게임 도메인 데이터 분석가 조민서입니다.

**10년 이상의 게임 유저 경험을 데이터로 번역합니다.**

인게임 경제·유저 행동 데이터를 직접 수집·모델링하여 "왜 가격이 올랐는가", "다음 패치는 어떻게 시장을 흔드는가"를 정량적으로 설명합니다. 게임을 소비하는 시각과 분석하는 시각을 동시에 가진 것이 가장 큰 무기입니다.

## 강점 요약

| | |
|---|---|
| **도메인** | 게임 인게임 경제, 유저 행동 패턴, 채용 시장 트렌드 |
| **무기** | 시계열 예측(ARIMA/Prophet) + Streamlit 실시간 배포 파이프라인 |
| **증명** | MAPE 1~5% 수준 예측 모델 → 대시보드 운영 중 |

## 분석 철학

* **데이터는 직접 수집합니다**: 공개 API와 크롤링으로 원천 데이터를 확보하고, 수집 구조 자체를 분석 설계의 일부로 다룹니다.

* **숫자로 말하고, 행동으로 연결합니다**: 분석의 끝은 인사이트가 아니라 의사결정입니다. "언제 사야 하는가", "어떤 스킬이 취업에 유리한가"처럼 행동으로 이어지는 결론을 목표로 합니다.
